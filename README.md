깃과 깃허브 첫 실습

# 깃 프로젝트 연습
- 폴더 생성시 > C드라이브 직접 생성 (원드라이브 X)
- 저장소 생성 후 "git init"
- 사용자 등록 > CD명령어로 저장소 이동 > git config user.name "~~" 
- 사용자 이메일 등록 > git config user.email "~~"

# 깃 커밋
- git add 파일명.확장자
- git -m "explanation of the file"
- git remote add origin 카피링크
